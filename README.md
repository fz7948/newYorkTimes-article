# newYorkTimes-article

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm start
```

## Infomation
New York Times 의 API : https://developer.nytimes.com/ 참조

Aticle Search API key를 발급받아 `.env.sample`을 참고하여 `.env`를 생성하고 키를 추가해주시면 됩니다.
